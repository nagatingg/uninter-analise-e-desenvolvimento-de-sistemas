-----História da Internet
-Arpanet 1969
    Rede militar que conectava quatro computadores no Oeste Americano
    Função: segurança de dados em posto de batalha - Guerra Fria
    Protocolo NCP (Network Control Protocol)
        FTP (File Transfer Protocol)
            Usado para transferência de arquivos, de forma rápida e segura
        DNS (Domain Name System)
            Usado para ligar nomes nos endereços, sem precisar decorar números
        SMTP (Simple Mail Transfer Protocol)
            Usado para emails
    (O que é um protocolo, conjunto de regras que estabelece uma comunicação)

-Protocolo TCP/IP - 1973
    (Protocolo de Transmissão mais importante de todos)
    Bob Kahn e Vint Cerf - primeiros a publicarem o termo internet
    TCP (Transmission Control Protocol) - vantagem quebra de mensagens em pacotes
    IP (Internet Protocol) - endereçamento de máquinas
    Tecnologias permitiram o aumento da rede
    Substituiu o NCP por completo em 1983

------Infraestrutura da Internet
-Cliente versus servidor
    -Cliente
        Computador ou serviço que requisita algo
        Informação disponível em outra máquina chamada servidor
        Exemplo:
            Navegadores web, aplicativos e jogos on-line

    -Servidor
        Computador dedicado a fornecer serviços
        Exemplo:
            Arquivos, páginas web, e-mail, banco de dados
        Softwares responsáveis por oferecer serviços, exemplo:
            Apache, Tomcat, Glassfish

-Como a internet funciona ( imagem salva na pasta )
    Seu computador > Modem > Router > Internet > Domain Name Server > Web Server
        Packet     <        <         <           <                    <
                > Request Path
                > Return Path

------------Principais Tecnologias
-HTML
    W3C (World Wide Web Consortium)
    Gerenciado por Tim Berners-Lee
    HTML1 - 1990
    HTML2 - 1995
    HTML3 - 1995
    HTML4 - 1997
    XHTML - 2004 (descontinuado)
    HTML5 - 2008 (html + css + javascript)

-HTML5
    Semântica
    Linguagem de marcação de hypertexto
    Não é uma linguagem de programação
    Documentação interpretada por navegadores

-CSS
    Estilo
    Folha de estilo em cascata
    Formata layout das páginas
    Atualmente na versão 3
    Também não é uma linguagem de programação

-Javascript
    Interatividade
    Manipulação de elemtnso de uma página
    Não é JAVA
    Esta é uma linguagem de programação interpretada
